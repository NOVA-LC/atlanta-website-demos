# Atlanta Website Demos

Pre-built demo sites for 10 Atlanta home-services prospects. Each demo has its own subfolder for independent Vercel deployment.

## Structure

```
/                                  → portfolio index
/national-leak-detection/          → demo 01
/flores-strawing/                  → demo 02
/marietta-radiator/                → demo 03
/pulliam-hvac/                     → demo 04
/atlanta-plumber-for-less/         → demo 05
/1st-class-hvac-contractors/       → demo 06
/atlanta-heating-and-cooling/      → demo 07
/jones-plumbing/                   → demo 08
/1st-class-hvac-atlanta/           → demo 09
/da-plumbing-septic/               → demo 10
```

Each subfolder is an independent static site (`index.html` only). Configure 10 separate Vercel projects, one per subfolder, to get 10 distinct URLs.

See `VERIFICATION_NOTES.md` for what data is verified vs research-derived.
